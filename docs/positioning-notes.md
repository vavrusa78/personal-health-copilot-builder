# Positioning notes

## Recommended public framing

Use language like:
- personal health copilot builder
- local-first nutrition and recovery tracking
- vendor-agnostic wellness system blueprint
- deterministic summaries with conservative health guardrails

## Avoid framing like

- AI doctor
- diagnosis copilot
- Oura assistant
- Apple Health sidekick
- autonomous medical advisor

## Recommended emphasis

The strongest value proposition is not one recap.
It is helping an agent build the whole system:
- ingestion
- canonical storage
- deterministic totals
- safe summaries
- optional presentation layers
